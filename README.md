# Custom-CSS-Injector-SPFx-for-Sharepoint-sites-pages
A SPFx in Nodejs that allows you to use CSS inside any sharepoint page/site. It's similar with the Wordpress CSS Customizer.  It needs Nodejs 18 (doesn't work with later versions) it need gulp to build and ship. A fully working sppgk exist in the repo at sharepoint\solution. 

For installation download directly the sppgk package from sharepoint\solution and upload it to the appcatalog of your sharepoint profile as admin. Insert the app on the page and then add from toolbar app of the sharepoint page directy to you page. In the "Edit Features" you will find the CSS customizer. 

Scope of this app is the same page that will inserted its time.
